🏆 實證醫學的證據等級

研究設計越嚴謹，偏差越少，證據力越強。NMA 位於金字塔頂端。

NMA / Systematic Review Level 1++
Meta-Analysis / RCT Level 1
世代研究 (Cohort Study) Level 2
病例對照研究 (Case-Control) Level 3
橫斷面研究 (Cross-Sectional) Level 4
個案報告 / 專家意見 (Case Report / Opinion) Level 5

⚖️ 傳統 Meta-Analysis vs. Network Meta-Analysis

📉 傳統 Meta-Analysis

  • 只能比較「兩種」治療方式
  • 僅有直接比較（A vs. B）
  • 無法排名多種療法優劣
  • 難以找出「最佳療法」
  • 呈現：森林圖 Forest Plot
A ←→ B（只能比較一對）

📈 Network Meta-Analysis

  • 可同時比較「多種」治療方式
  • 直接 + 間接比較都納入
  • 計算 SUCRA 給出排名
  • 能找出「最佳療法」
  • 呈現：網絡圖 + 排名圖
A ←→ B ←→ C ←→ D ←→ E（全部一起比！）
💡 關鍵概念：間接比較 — 即使 A 和 D 從未在同一個 RCT 中比較過，只要 A 有與 B 比較、B 有與 D 比較，NMA 就能推算出 A vs. D 的相對效果！

🕸️ 網絡幾何圖 (Network Geometry)

圓圈 = 治療方式（大小代表研究數量），連線 = 兩者有直接比較的 RCT

節點越大 = 納入研究數越多；線段越粗 = 直接比較研究越多

🗺️ NMA 分析五大步驟

1

建立網絡幾何圖 Network Geometry

釐清哪些治療方案有直接比較的研究，畫出網絡圖，設定基準對照組（Reference）。

network map
2

檢定不一致性 Testing for Inconsistency

NMA 的重要前提假設：直接比較與間接比較的結果必須一致。分為整體（Global）和局部（Local）兩種檢定。若 p > 0.05，無法拒絕虛無假說 → 一致性可接受，繼續分析。

network meta inconsistency network sidesplit all
3

建立效應值圖表與位階表 Effect Size Plots & League Table

產出網絡森林圖（Network Forest Plot）及區間圖（Interval Plot），呈現各組別之間的效應值（ES）及 95% 信賴區間。

network forest intervalplot, eform
4

決定治療的相對排名 Relative Rankings (SUCRA)

計算 SUCRA（累積排名曲線下面積）值，數值越高 → 越可能是最佳療法。本範例中 D 組（Topical）SUCRA = 1.0，達到最佳排名的機率為 98.3%。

network rank min/max
5

檢查出版偏差 Publication Bias

繪製網絡漏斗圖（Network Funnel Plot），檢視標準誤（SE）與效應值是否對稱分布。若漏斗形狀對稱 → 出版偏差小，分析結果較可信。

netfunnel diff se t1 t2, random

🔭 出版偏差檢定 — 網絡漏斗圖

什麼是出版偏差？
有陽性（顯著）結果的研究更容易被發表，導致 meta-analysis 結果被高估。

如何用漏斗圖檢查？
X 軸 = 效應值大小，Y 軸 = 標準誤（SE）大小。研究越精確（樣本數大）→ 標準誤越小 → 點越靠近頂端。

如何判讀？

  • ✅ 對稱分布 → 出版偏差小，結果可信
  • ❌ 不對稱 → 可能有出版偏差
Stata 指令：
netfunnel diff se t1 t2, random bycomparison

💡 總結重點

Network Meta-Analysis 核心精華

  • NMA 是傳統 Meta-Analysis 的延伸，可以同時比較多種療法並給出排名
  • 透過直接 + 間接比較，即使兩種療法沒有直接 RCT，也能推算相對效果
  • 五大步驟：網絡圖 → 一致性檢定 → 效應值圖 → SUCRA 排名 → 出版偏差
  • SUCRA 越高 = 越可能是最佳療法（本例：D 組局部使用 = 1.0，最佳）
  • 單一 RCT 有隨機誤差風險，NMA 整合多研究，證據力更強、更客觀
  • 分析時需注意：研究種類、品質、訊息強度，勿盲目信任結果
